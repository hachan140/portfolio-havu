Context

Previously, when creating a store on the ShopBase ecosystem platform, store owners would design their stores using the theme editor tool. However, this tool had limitations that hindered sellers from customizing their websites flexibly. It lacked standout features, essentially making long-term competitiveness challenging. The ShopBase Website Builder toolkit allows users to drag and drop blocks to easily and flexibly create a website. Without the knowledge of coding, users can build a professional and aesthetically pleasing store using the Website Builder. Learn more about Web Builder.

Responsibilities

One of the crucial blocks for a new e-commerce website is the Tab/Accordion. This block consistently appears on the product page, used to display information such as product description, shipping details, and warranty information. My responsibility is to build a Tab/Accordion block that users can drag and drop onto any desired website/page.

After surveying over 300 popular e-commerce websites, I identified that sellers have a need for using Tabs and Accordions with various complex layouts. Additionally, users often want the same block to be displayed as a tab on desktop but as an accordion on mobile.

Solution

I developed a Tab & Accordion as a component that can be inserted, moved, edited, and deleted within the web builder.

The Tab & Accordion includes a setting called Display = Tab/Accordion. If viewed on mobile, the display is set to accordion. If viewed on desktop, the display is set to tab. This setting is customizable based on the user's preferences.

I revised the structure of the blocks in the web builder to enhance the capability of making blocks contain other blocks. Specifically, Tab & Accordion will have a layout consisting of multiple rows and columns and can contain other blocks, including other tabs and accordions.
